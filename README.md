## WhatsApp Clone

Es una aplicación que simula un la interfaz de WhatsApp.
Incluye:
-Lista de contactos
-Vista de chats
-Envío y eliminación de mensajes
-Navegación entre contactos

## Tecnologias utilizadas
-React
-React Router
-Vite
-HTML5
-CSS
-JavaScript
